# OKR 示例库

## 一、产品类 OKR

### ❌ 不好的例子

```
O：提升产品体验
KR1：优化首页设计
KR2：修复已知 bug
KR3：用户反馈更好了
```

**问题分析：**
- O 缺乏方向感，"提升"太模糊
- KR1 是任务，不是结果
- KR2 是任务，不是结果
- KR3 不可量化，"更好了"无法衡量

### ✅ 好的例子

```
O：打造令用户惊艳的首页体验
KR1：首页加载时间从 3s 降低到 1.5s
KR2：首页跳出率从 45% 降低到 30%
KR3：首页功能可用性测试满意度达到 4.5/5 分
```

**优点：**
- O 有方向感和价值感
- 每个 KR 都可量化
- KR 与 O 强相关

## 七、转化技巧总结

### 从任务到结果

| ❌ 任务型 KR | ✅ 结果型 KR |
|-------------|-------------|
| 优化首页设计 | 首页跳出率从 45% 降到 30% |
| 修复 bug | 线上故障率从 5% 降到 1% |
| 多投广告 | 获客成本从 50 元降到 30 元 |
| 提升用户体验 | NPS 从 30 提升到 45 |

### 从模糊到具体

| ❌ 模糊描述 | ✅ 具体量化 |
|------------|------------|
| 提升性能 | 响应时间从 500ms 降到 200ms |
| 增加收入 | ARR 从 500w 增长到 1200w |
| 改善体验 | 满意度从 3.8 提升到 4.5 |
| 提高效率 | 人均产出从 10 单/天提升到 15 单/天 |

### 量化公式的应用

**公式：** `指标 + 基线值 + 目标值`

示例：
- 加载时间：3s → 1.5s
- 用户数：10w → 15w（+50%）
- 转化率：2% → 4%（翻倍）
- 成本：50元 → 30元（-40%）
