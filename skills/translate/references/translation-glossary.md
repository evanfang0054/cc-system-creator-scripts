## 不翻译的术语

以下术语保持英文原文：

- React, Vue, Angular, Next.js, Nuxt.js
- TypeScript, JavaScript
- npm, yarn, pnpm
- webpack, vite, rollup
- ESLint, Prettier
- Git, GitHub, GitLab
- Docker, Kubernetes
- Linux, macOS, Windows
- CSS, HTML, JSON, YAML
- SQL, NoSQL
- OAuth, JWT
- CI/CD

## 翻译原则

1. **技术名词优先保留英文**：如 API、SDK、CLI
2. **有通用中文译名的使用中文**：如 仓库、分支、提交
3. **首次出现可加括号注释**：如 代理（Agent）
4. **保持一致性**：同一文档中术语翻译保持统一
5. **代码不翻译**：变量名、函数名、命令等保持原样
